# 🌌 Interactive Solar System - BTI Project

Simulador interativo do sistema solar desenvolvido como parte do meu portfólio técnico. O projeto permite a exploração visual dos astros, com navegação livre via mouse e informações detalhadas de cada planeta.

## 🚀 Funcionalidades

* **Navegação Dinâmica:** Arraste o universo com o mouse para explorar regiões distantes.
* **Interface Responsiva:** Modal de boas-vindas com instruções de uso.
* **Dados Técnicos:** Painel lateral que exibe informações e imagens de 9 astros (Sol a Plutão).
* **Simulação de Órbitas:** Movimentos orbitais sincronizados via CSS Animations.
* **Foco em UX:** Atalhos de teclado (ESC) para fechar painéis e pausa automática das órbitas durante a leitura.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica dos elementos celestes.
* **CSS3:** Estilização avançada, filtros de brilho (box-shadow) e animações de translação.
* **JavaScript (Vanilla):** Lógica de manipulação de DOM, cálculos de coordenadas para o sistema de drag (arraste) e tratamento de erros de mídia.
